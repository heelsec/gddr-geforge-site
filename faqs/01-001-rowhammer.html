What's the Rowhammer bug? Rowhammer is a DRAM reliability error where repeated accesses to a DRAM row can cause bits to flip in neighboring rows. Rowhammer can be used to violate many security policies employed in a system. To combat this, DRAM manufacturers all implement hardware level mitigations in their memory. Conducting a successful Rowhammer attack requires the attacker to reverse engineer these proprietary mitigations.
